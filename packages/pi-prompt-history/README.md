# Pi Prompt History

[![npm version](https://img.shields.io/npm/v/@zigai/pi-prompt-history.svg?color=blue)](https://www.npmjs.com/package/@zigai/pi-prompt-history)
[![npm downloads](https://img.shields.io/npm/dm/@zigai/pi-prompt-history.svg)](https://www.npmjs.com/package/@zigai/pi-prompt-history)
[![license](https://img.shields.io/npm/l/@zigai/pi-prompt-history.svg)](../../LICENSE)

This Pi extension adds cross-session prompt history to the prompt editor.

## Install

```sh
pi install npm:@zigai/pi-prompt-history
```

## Features

- Loads recent prompt history from current and previous sessions for the same working directory.
- Keeps prompt recall scoped to the project you are working in.
- Preserves Pi app-level editor keybindings while augmenting the editor history.
